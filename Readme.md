# @davingm/roobert

Roobert Fonts untuk Nuxt 4 pada Davingm UI

## Instalasi

```bash
pnpm add @davingm/roobert
```

## Pemakaian

```ts
export default defineNuxtConfig({
  modules: ['@davingm/roobert']
})
```

## Gunakan !
```css
body {
  font-family: "Roobert", sans-serif;
}
```
